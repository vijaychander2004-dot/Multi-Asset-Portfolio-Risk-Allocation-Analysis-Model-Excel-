# Multi-Asset Portfolio Risk & Allocation Analysis Model (Excel)

## Project Overview
This project builds an Excel-based multi-asset portfolio model to evaluate portfolio risk and return across different allocation strategies.

The model calculates expected portfolio returns and risk using weighted averages and variance–covariance logic.

It compares multiple portfolio allocation strategies to analyze their impact on risk and return.

---

## Key Features
- Expected portfolio return calculation
- Portfolio variance and risk estimation
- Variance–covariance portfolio risk modeling
- Comparison of allocation strategies
- Built-in consistency checks for portfolio weights

---

## Allocation Strategies Compared
- Brown Portfolio
- Green Portfolio
- Mixed Portfolio

Each strategy uses different asset weight combinations to evaluate the trade-off between expected return and portfolio risk.

---

## Tools Used
- Microsoft Excel
- Portfolio theory concepts
- Variance–covariance risk modeling
